## Volume Management

### Backup Ollama models

```bash
docker run --rm \
  -v nongtime_ollama_data:/data \
  -v $(pwd):/backup \
  alpine tar czf /backup/ollama-backup.tar.gz -C /data .
```

### Restore Ollama models

```bash
docker run --rm \
  -v nongtime_ollama_data:/data \
  -v $(pwd):/backup \
  alpine sh -c "cd /data && tar xzf /backup/ollama-backup.tar.gz"
```
